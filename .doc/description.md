Palette Editor helps you build complete color systems inside Figma. Pick a few key colors and get full, perceptually even tonal scales you can export straight to variables, styles, or canvas swatches.

**What you can do**

- Start from key colors — add them by hand, generate harmonious ones, pull them from a canvas selection, or extract a palette from any image (drop, paste, upload, or link).
- Auto-named colors — every color is named for you until you pin your own name.
- Tonal scales — each key color becomes a shade ramp on a configurable 0–1000 scale (2–26 steps), generated through a gradient. You can fine-tune it stop by stop, with chroma compensation for natural mid-tones.
- Adjust shade stops — if you need so, or let them be evenly distributed.
- Perceptual color — work in OKLCH / LCH (plus HSL, HSV, RGB), so scales stay even and chroma behaves sensibly across lightness; out-of-gamut colors are mapped, not clamped to nonsense.
- Multiple palettes — keep one shared palette in the file (the single source for your variables and styles, visible to everyone editing it) plus have a private library of palettes available across every file. Switch between them, experiment freely, and import colors from one palette into another.
- Export — create variables, color styles, or place a grid of swatches on the canvas.
